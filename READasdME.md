#Face-Overlay

##Glass overlay on face using python/opencv/dlib

###Requirements
1.Python 3

And the following libraries

2.opencv
3.dlib
4.numpy (Required dependancy for previous)
5.scipy

###Setup Instructions
*Clone Repository
`git clone https://github.com/spell1612/Face-Overlay.git`
*cd into directory
`cd Face-Overlay`
*Run `python GlassFace.py`
*Press 'q' exit
